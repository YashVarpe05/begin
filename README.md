## 🛠️ Getting Started

Follow these steps to set up the project locally:

1. **Clone the repository**:
    ```bash
    git clone https://github.com/YashVarpe05/begin.git
    cd begin
    ```

2. **Install dependencies**:
    ```bash
    npm install
    ```

3. **Start the development server**:
    ```bash
    npm run dev
    ```

4. **Build for production**:
    ```bash
    npm run build
    ```
